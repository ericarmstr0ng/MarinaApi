# Marina API
## GETTING STARTED ##
- Go to https://project-loki-254500.appspot.com and click ‘Authorize’
- Log in to google
- User Id and JWT will display
- Use User Id for GET users/:user_id call 
- Use JWT for Bearer token

## Usage ##
- Make API calls as listed in MarinaAPI_Manual
